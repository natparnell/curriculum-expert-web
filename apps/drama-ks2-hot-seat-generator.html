Teacher notes: running a hot seat (KS2)

Characterisation, status and role. Hot-seating asks one pupil to answer as a character while the class probes them. This app does three jobs:

  • Role: the pupil stays in character, using "I" and answering as that person, never as themselves.
  • Status: high-status characters speak with confidence and command; low-status characters are tentative, polite or fearful. The status bar and badge make this explicit. Use "Try a different status" to show how the same role shifts when status changes.
  • Characterisation: good answers add specific detail (a name, a place, a feeling) that fits the character.

Run it on the whiteboard: one pupil sits in the hot seat, press "Next question", the pupil answers, then the class votes "Stayed" or "Slipped". The feedback prompts a quick discussion. No real pupil data is stored; all names are fictional.
