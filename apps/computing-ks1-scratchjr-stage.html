The cat starts on the green flag. Each move block slides it one square. The grid is 5 squares wide and 5 squares tall.

  • Predict first: where will the cat end up?
  • Run the program and watch.
  • If it is wrong, that is a bug. Add, remove or reorder blocks.
  • The little ✖ on a block deletes it.
