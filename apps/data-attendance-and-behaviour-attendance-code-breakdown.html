Attendance Code Breakdown

Teacher aim: pupils decompose a cohort's absence by statutory DfE attendance code, separating authorised illness from unauthorised holidays and truancy, so they can read where the real attendance problem sits for each year group.

The DfE attendance codes

Every half-day session a pupil misses is logged with a single letter. The letter decides whether the absence is authorised, unauthorised or counted as present / approved.

Step 1: Explore the breakdown

Pick a year group. The bar splits its lost sessions by code so you can see why pupils were absent, not just how often.
