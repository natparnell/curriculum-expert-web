How to Read a 4-Figure Grid Reference

Step 1: Read ALONG the bottom first (the Easting). Find the number on the horizontal axis.

Step 2: Read UP the side (the Northing). Find the number on the vertical axis.

Remember: "Along the corridor, up the stairs!" Eastings always come before Northings.
