Sound Sampling: Digitising Analogue Sound

Teaching aim: understand how a continuous analogue sound wave is converted into a digital signal by sampling, how sample rate and sample resolution (bit depth) affect playback quality and file size, and how to calculate a sound file size using the formula sample rate × bit depth × duration.

File size formula (OCR J277)
file size (bits) = sample rate × bit depth × duration × channels = 8000 × 4 × 10 × 1 = 320,000 bits ÷ 8 = 40,000 bytes ≈ 39.06 KB (÷ 1024)

Why it matters: the quality vs size trade-off. Every extra unit of quality costs storage and bandwidth.

Raising the sample rate takes more snapshots each second, so fast changes in the wave are captured and high notes are not lost. Raising the bit depth gives more amplitude levels, so each sample is rounded by a smaller amount and quiet detail survives. Both make the reconstructed wave hug the original more closely, but the file size grows in direct proportion to each.

That is the engineering decision: a podcast can use a low rate and depth to keep files small, while a music master uses CD quality (44,100 Hz, 16-bit) or higher because the listener will notice. Try the presets and watch the file size jump.

How sampling works (the four steps)

1. Capture: a microphone produces a continuous analogue voltage that mirrors the sound pressure.

2. Sample: an analogue-to-digital converter measures the amplitude at fixed intervals set by the sample rate.

3. Quantise: each measurement is rounded to the nearest level allowed by the bit depth, then stored as a binary number.

4. Reconstruct: on playback the stored values rebuild a stepped approximation of the wave. The closer it matches the original, the better the sound.

Exam-style questions (reveal the model answer)

Q1Define the term sample rate and state its unit. [2]

The sample rate is the number of samples taken per second when an analogue signal is recorded. It is measured in hertz (Hz); for example CD audio uses 44,100 Hz.

Q2Explain how increasing the sample resolution (bit depth) affects the recording. [3]

A higher bit depth gives more possible amplitude levels (2 to the power of the bit depth). Each sample can therefore be rounded to a value closer to the true amplitude, reducing quantisation error. This makes playback quality more accurate, but increases the file size because more bits are stored per sample.

Q3A 30 second mono clip is sampled at 44,100 Hz with a bit depth of 16. Calculate the file size in megabytes. [4]
file size (bits) = sample rate × bit depth × duration = 44100 × 16 × 30 = 21,168,000 bits bytes = 21,168,000 ÷ 8 = 2,646,000 bytes MB = 2,646,000 ÷ 1,048,576 ≈ 2.52 MB

Answer: approximately 2.52 MB (or 2.65 MB if 1 MB is treated as 1,000,000 bytes). Always check which convention the question expects.

Q4A sound engineer wants smaller files for a spoken-word podcast. Suggest two changes and the trade-off. [4]

Sensible changes: lower the sample rate (for example to 22,050 Hz, as speech has no very high frequencies) and reduce the bit depth (for example to 8 bits), and record in mono rather than stereo. Each change reduces the data stored, so the file is smaller.

The trade-off is reduced quality: a lower rate cannot capture high frequencies and a lower bit depth introduces more quantisation noise, but for speech this loss is usually acceptable.

Q5Why does a stereo file take more storage than a mono file of the same length and settings? [2]

Stereo stores two separate channels (left and right) instead of one, so twice as many samples are recorded. The file size formula is multiplied by the number of channels, doubling the storage compared with mono.

Key terms (OCR J277)

TermMeaning
AnalogueA continuous signal that can take any value, such as a real sound wave.
SamplingMeasuring the amplitude of an analogue signal at regular intervals.
Sample rateHow many samples are taken per second, measured in hertz (Hz).
Sample resolutionThe number of bits used to store each sample; also called bit depth.
QuantisationRounding each sample to the nearest available amplitude level.
ReconstructionRebuilding a stepped approximation of the wave from the stored samples.
