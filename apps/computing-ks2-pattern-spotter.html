🔍 Pattern Spotter

Computing • Key Stage 2 • Computational thinking

Teacher aim: pupils use decomposition and pattern spotting to find a repeating rule in a growing tile pattern, then write that rule as one general formula. This is abstraction made visible.

Step 1: Spot the pattern

Look at how this tile pattern grows. Each picture is one step. How many orange tiles are added each time?

Tip: count along the bottom row and the side. The pattern grows in a steady way.

Step 2: Predict the next step

Decomposition: break the problem down. Use the pattern you spotted to work out the hidden step, shown with a dashed blue outline above.

Count the tiles in each step, then predict the hidden one.
How does this help? (for teachers)
Pupils first decompose the picture into rows they can count, then spot the pattern in how the count changes, then reason logically to predict an unseen step. Getting the next step right unlocks the abstraction task below.

Step 3: Make the rule (abstraction)

Abstraction means writing one rule that works for every step, not just the ones you can see. Build the rule for the number of tiles in step n.
