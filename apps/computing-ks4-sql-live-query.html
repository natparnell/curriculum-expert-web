SQL Live Query

Teacher aim: pupils build SELECT … FROM … WHERE queries against a school database table of records and see the result set filter instantly, then connect Boolean conditions to logic gates and truth tables.

1 Build a query

Choose the columns to SELECT, then add a WHERE condition to filter the rows. The table on the right updates as you go.

2 Result set

3 Query challenges

Set your query so the result set exactly matches each target. Solve them in any order; the card ticks the moment your result is right.

4 From WHERE to Boolean logic

A WHERE condition is a Boolean expression: each comparison is true (1) or false (0), and AND / OR combine them exactly like logic gates. Flip the inputs and predict the output.
