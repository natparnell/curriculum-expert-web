    Bounds and error bars

    A measurement that has been rounded could really be anything within half a unit of accuracy each side. Set a value and its accuracy to see the error interval as a bar, then test yourself.

    Lower bound (smallest it could be)
    Upper bound (everything below this)

    The half-unit either side is half of the accuracy. The upper bound is written with the "less than" idea: the true value can get as close as you like to it but never reach it.
