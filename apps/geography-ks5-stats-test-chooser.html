How the calculation works

Rank each variable separately from highest to lowest. Find the difference in ranks (d) for each site, square it (d²), then total the column (Σd²).

Apply the formula rs = 1 − ( 6 Σd² ) / ( n³ − n ), where n is the number of paired sites.

How the calculation works

For each cell, the expected value E = (row total × column total) / grand total. This is the count you would expect if there were no association.

For every cell compute (O − E)² / E, then total them to give χ². Degrees of freedom = (rows − 1) × (columns − 1).
