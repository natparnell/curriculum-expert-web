💡 Lab notes

Drag the Repeat block into your program to begin. Then add a Move and a Turn inside it.
What do the words mean?

Sequence: instructions run in order, top to bottom.

Repetition: a loop repeats the inside blocks a number of times. More repeats make the spiral longer.

Selection: an if block makes a choice. The blocks inside only run when the condition is true.

Variable: a labelled box that stores a number you can change, like a score.

Tips for a tidy spiral

Keep Move small (about 4 to 8) and Turn a bit under or over 90 degrees. Then raise the Repeat count and watch the spiral tighten.
