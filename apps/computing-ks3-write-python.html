Variables, arithmetic and types
A variable stores a value. Assign with =. Numbers can be whole (int) or decimal (float). Operators: + - * /, plus floor division //, remainder % and power **. 
age = 13
height = 1.62
total = age * 2 + 1     # 27
half  = 7 // 2          # 3  (floor division)
rest  = 7 % 2           # 1  (remainder)
cube  = 2 ** 3          # 8
Strings, str() and f-strings
Text goes in quotes. Join text with + (both sides must be strings). Turn a number into text with str(), or use an f-string with { } placeholders. 
name = "Sam"
print("Hi " + name)             # Hi Sam
score = 9
print("Score: " + str(score))   # Score: 9
print(f"{name} scored {score}") # Sam scored 9
Input and output
print(...) shows a line in the console. input("prompt") reads one line of text. Wrap it in int(...) or float(...) to get a number. In this app the input lines come from the Inputs box on the right. 
name = input("Name? ")
age  = int(input("Age? "))
print(f"Hello {name}, next year you are {age + 1}")
Selection: if / elif / else
Run code only when a condition is true. Compare with == != < > <= >= and combine with and, or, not. Indent the block by four spaces. 
mark = 72
if mark >= 70:
    print("Distinction")
elif mark >= 50:
    print("Pass")
else:
    print("Try again")
Iteration: for and while
for i in range(n) repeats n times with i = 0,1,2,... range(a, b) counts from a up to (not including) b. A while loop repeats while its condition is true. 
for i in range(1, 4):
    print(i)            # 1, 2, 3

count = 3
while count > 0:
    print(count)
    count = count - 1   # 3, 2, 1
Lists: indexing, append, len
A list holds many values. Index from 0. Add with .append(...) and count with len(...). 
scores = [4, 8, 2]
print(scores[0])        # 4
scores.append(10)
print(len(scores))      # 4
total = 0
for s in scores:
    total = total + s
print(total)            # 24
