CAD/CAM explained: from screen to workshop

What CAD and CAM mean

Computer-aided design (CAD) is using software to create accurate 2D drawings or 3D models of a product. Computer-aided manufacture (CAM) is using a computer to control the machine that makes the part. CAD produces the geometry; CAM converts that geometry into the instructions the machine follows.

Advantages of CAD

  • Accuracy: dimensions are exact and repeatable, far better than hand drawing.
  • Easy editing: a change to one dimension updates the whole drawing without redrawing from scratch.
  • Sharing: files can be sent instantly to clients, suppliers or other designers anywhere.
  • Simulation and testing: stress, fit, motion and material use can be checked on screen before any material is cut, reducing waste and cost.
  • Standard parts and libraries: common components can be reused, saving time.

How a CAD drawing becomes a toolpath

A post-processor reads the CAD geometry and generates a toolpath: the exact route the cutting head takes, written as machine code (often G-code). It plans the order of operations, where to plunge or pierce, the cutting moves along each outline, and the rapid travel moves in between. Engraving is usually done before cutting, and internal holes before the outer profile, so the part stays held in the sheet until last.

Tolerance

Tolerance is the permitted variation on a dimension, written as a plus or minus value (for example 40 mm ±0.1 mm). No process is perfectly precise, so a tolerance states how much error is acceptable while the part still works. A tighter tolerance costs more time and money; a looser tolerance is cheaper but may not fit. Designers also allow for the kerf (the width of material the cutter removes) so that slots and joints end up the right size.

Nets and development

A net (or development) is a 3D form unfolded into a flat shape. It lets a 3D product be cut from flat sheet then folded or assembled. The net generator in this studio develops the six faces of a box and adds finger joints so the faces interlock.

CAD/CAM and production volume

  • One-off / prototype: CAD lets you make a single accurate part quickly; CAM cuts it directly with no tooling cost.
  • Batch: the same CAD file is reused to nest many parts on a sheet, so a batch is cut efficiently with consistent quality.
  • Mass production: CAD models drive injection-mould or die tooling; the same digital master guarantees every product is identical.

Exam-style questions

Q12 marks

State the difference between CAD and CAM.

Indicative content: CAD (computer-aided design) is using software to create and edit accurate 2D drawings or 3D models of a product (1). CAM (computer-aided manufacture) is using a computer to control the machine that actually makes the part from the CAD data (1).
Q24 marks

Explain two advantages of using CAD when designing a product.

Any two, explained (1 + 1 each):
  • Accuracy: dimensions are exact and repeatable, so parts fit together first time.
  • Easy editing: changing one dimension updates the whole drawing, so modifications are fast and there is no need to redraw.
  • Sharing: files can be emailed instantly to suppliers or clients anywhere in the world.
  • Simulation: the design can be tested on screen for fit, strength or material use before any material is cut, cutting waste and cost.
Q33 marks

Describe how a 2D CAD drawing is turned into the instructions a laser cutter follows.

Indicative content: the CAD geometry is passed to a post-processor / CAM software (1), which generates a toolpath (the route the cutting head takes) as machine code such as G-code (1). The toolpath sets the order of operations: engraving and internal holes first, then the outer profile last, with rapid travel moves between features (1).
Q42 marks

A slot is dimensioned 40 mm ±0.1 mm. Explain what the ±0.1 mm means and why a designer uses it.

Indicative content: it is the tolerance: the slot may be made anywhere between 39.9 mm and 40.1 mm and still be acceptable (1). It is used because no process is perfectly precise, so the tolerance states the permitted variation while the part still fits and functions (1).
Q54 marks

A designer needs to make 5,000 identical components. Explain how CAD/CAM supports this scale of production compared with making a single prototype.

Indicative content (any sensible developed points):
  • The single CAD master guarantees every component is identical, giving consistent quality across the batch/mass run (1+1).
  • The same CAD file drives the tooling (for example an injection mould or die), so once the tool is made the parts are produced quickly and at low unit cost (1+1).
  • For a one-off, CAM cuts straight from the file with no tooling cost, which is cheaper for a single part but not efficient at 5,000 (1+1).
