▶ Bonus: change the subject of the quadratic formula

Completing the square is exactly how the quadratic formula is built. Starting from ax² + bx + c = 0, you can rearrange step by step to make x the subject:

1 Divide by a: x² + (b/a)x + c/a = 0

2 Complete the square: (x + b/2a)² − (b/2a)² + c/a = 0

3 Isolate the bracket: (x + b/2a)² = (b² − 4ac) / 4a²

4 Square root both sides, then subtract b/2a: the result is the quadratic formula. So "change the subject" and "completing the square" are the same skill applied to letters instead of numbers.
