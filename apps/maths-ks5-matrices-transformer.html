How it works: the mathematics behind the tool

A matrix as a transformation

A 2×2 matrix M = [[a, b],[c, d]] sends the basis vector i = (1, 0) to the first column (a, c), and j = (0, 1) to the second column (b, d). Because the transformation is linear, every point (x, y) maps to (ax + by, cx + dy): the whole plane is determined by where i and j land. On the canvas the blue arrow is the image of i and the pink arrow is the image of j.

The determinant as an area scale factor

The determinant det(M) = ad − bc is the factor by which areas are multiplied. The unit square has area 1, so its image (a parallelogram) has area |det(M)|. The sign records orientation: a positive determinant preserves the sense of rotation (anticlockwise stays anticlockwise), a negative determinant reflects it (a reflection or an odd number of reflections), and det(M) = 0 collapses the plane onto a line, destroying area.

Standard transformations

  • Rotation by θ anticlockwise: [[cosθ, −sinθ],[sinθ, cosθ]], det = 1.
  • Reflection in the line y = x·tanθ: [[cos2θ, sin2θ],[sin2θ, −cos2θ]], det = −1.
  • Enlargement, scale factor k about the origin: [[k, 0],[0, k]], det = k².
  • Shear parallel to the x-axis, factor k: [[1, k],[0, 1]], det = 1 (area preserved).

Composition: order matters

If you apply M first and then N, the combined matrix is the product NM (apply the right-most matrix first). Matrix multiplication is generally not commutative, so NM ≠ MN: rotating then reflecting is usually a different transformation from reflecting then rotating. The determinant of a product is the product of the determinants: det(NM) = det(N)·det(M).

The inverse and solving systems

The inverse undoes the transformation: M⁻¹ = (1/det) [[d, −b],[−c, a]]. It exists only when det(M) ≠ 0; a singular matrix has squashed the plane onto a line and cannot be reversed. The same inverse solves a pair of simultaneous equations written as Ax = b: the unique solution is x = A⁻¹ b, which exists precisely when det(A) ≠ 0. Geometrically, you are asking which input vector x the matrix A maps to b.
