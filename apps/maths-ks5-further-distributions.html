Poisson Distribution & Chi-squared Tests

The Poisson model

A count X follows a Poisson distribution, written X ~ Po(λ), when events occur singly, independently, and at a constant average rate λ over an interval.

P(X = r) = e−λ λr / r!
  • E(X) = λ and Var(X) = λ. If real data has variance much larger than its mean, Poisson is a poor fit (over-dispersion).
  • If X ~ Po(λ1) and Y ~ Po(λ2) are independent, then X + Y ~ Po(λ1 + λ2).
Worked example: calls to a helpline

A support helpline receives calls at a mean rate of 3 per hour, modelled as Po(3).

(a) P(exactly 2 calls in an hour) = e−3 3² / 2! = 0.2240.

(b) P(at most 2) = P(0) + P(1) + P(2) = 0.0498 + 0.1494 + 0.2240 = 0.4232.

(c) P(more than 2) = 1 − 0.4232 = 0.5768.

Set the slider to λ = 3 and r = 2 to reproduce these figures.

Why the limit works

Take X ~ B(n, p) with p = λ/n. As n → ∞ with λ fixed:

B(n, λ/n) → Po(λ)
  • The binomial variance is np(1 − p) = λ(1 − p). As p → 0 this tends to λ, matching the Poisson variance.
  • This is the standard rule of thumb for the Poisson approximation to the binomial: use it when n is large and p is small (commonly n > 50 and p < 0.1).
  • The approximation lets you model rare events (defective items, accidents, decay) without large factorials.

The goodness-of-fit test

χ² = Σ (O − E)² / E
  • Compute the test statistic by summing (O − E)² / E over every category.
  • Degrees of freedom = (categories) − 1 − (parameters estimated). For the fair die nothing is estimated, so ν = 6 − 1 = 5.
  • If χ² > critical value, the result lies in the rejection region: reject H₀. Otherwise there is insufficient evidence to reject H₀.
  • For a fitted Poisson or binomial model you also subtract 1 for the estimated mean, and pool any class with expected frequency below 5.
Worked example: marginal evidence

Observed counts 16, 18, 20, 25, 19, 22 give χ² = (16−20)²/20 + … = 2.50.

With ν = 5 the 5% critical value is 11.07. Since 2.50 < 11.07 we do not reject H₀: the data are consistent with a fair die.

Test of independence

E = (row total × column total) / n
  • For an r × c table the degrees of freedom are (r − 1)(c − 1). Here (3 − 1)(3 − 1) = 4.
  • Each expected value assumes the row and column classifications are independent.
  • Compare χ² = Σ(O − E)²/E against the critical value. If it exceeds the critical value, reject H₀ and conclude the variables are associated.
  • Every expected frequency should be at least 5 for the χ² approximation to be reliable.
