Decompose-It 🧩

Teacher aim: Pupils decompose a maze-solving goal into smaller sub-goals on sticky notes, assemble them into a program in the right order, then run a step-through debugger to find and fix mistakes. Covers designing, writing and debugging programs to solve problems.

1Decompose the goal

The big goal is: get the robot to the green flag. Break it down. Drag the sub-goal sticky notes into the program in the order the robot should follow them. Tap a note then press Enter to pick it up with the keyboard.

Sub-goal notes

    Tip: you will not need every note. Choosing only the steps you need is part of decomposing.

      2Run and debug

      Watch the robot follow your program one line at a time. If it hits a wall or stops short, that is a bug. Find which line is wrong, fix the order, and run it again.
