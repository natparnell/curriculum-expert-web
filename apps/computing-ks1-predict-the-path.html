Computing • KS1 • Logical reasoning

Predict the Path

Teacher aim: pupils read a short command sequence and predict where the robot will stop, then check by running it. Builds logical reasoning: a program does the same thing every time it runs.

1. Read the program

Robi the robot starts on the yellow square facing the way the arrow points. Read each command in order.

2. Predict the end square

Tap the square where you think Robi will stop. You can change your mind until you press Run. Use arrow keys to move the highlight, then press Enter to choose.

Squares are named by column then row, for example B3.

3. Vote, then run

Pick a square first
Choose where Robi will stop, then press Run to find out.

Teacher tip: pause on the prediction and ask "why there?" before running. When the reveal differs, replay it one step at a time and talk through the command that surprised them. Same program, same start = same finish, every time.
