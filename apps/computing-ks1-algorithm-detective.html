🔎 Algorithm Detective

Computing · Key Stage 1 · Algorithms: precise step-by-step instructions

Detective on duty: Robo-Ray
Teacher aim: Pupils learn that an algorithm is an ordered, unambiguous set of instructions. They debug a muddled "get dressed" algorithm, predict what each step does, and reorder the steps until Robo-Ray is dressed correctly.

📋 The wonky algorithm

These get-dressed steps are in the wrong order. Drag a step to move it, or use the keyboard (see tip below). Then press Run the algorithm to see what Robo-Ray does.

    Keyboard: Press Tab to a step, then Space or Enter to pick it up. Use the ↑ and ↓ arrows to move it. Press Space or Enter again to drop it.

    🤖 Robo-Ray follows the steps

    🤖
    Robo-Ray is waiting for instructions.
    Steps in the right place: 0 of 5
    Put the steps in a sensible order, then press Run. A good algorithm has no muddles: every step must come before the steps that need it.
