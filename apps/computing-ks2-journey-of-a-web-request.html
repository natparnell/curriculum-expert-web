📦 Journey of a Web Request

How a web page travels from a server, across the internet, to your screen.

Teacher aim: Pupils understand that the World Wide Web is delivered over computer networks, and can name the job of the device, router, ISP and web server in a request and response. (KS2 Computing: computer networks, the internet and the World Wide Web.)

1. Send a request and watch the packet travel

Picking a page changes what the web server sends back.
Press "Send request" to start the journey.
Device Router ISP Server Packet of data

Tap, click or use Tab and Enter on any blue, green, pink or orange box to learn its job.

🔍Click a hop to learn about it

Each box is a "hop" on the journey of your web request.

A web request travels left to right (your question), then the page travels back right to left (the answer). Select any box above to find out what it does.
