Tree Diagram Builder

Teaching aim: probabilities multiply along the branches and add across the outcomes. Adjust a branch and watch every outcome update, then compare with and without replacement.

1. Choose a two-stage experiment

Each scenario draws two items in turn. Switch replacement on or off to see conditional probability change the second set of branches.

2. Build and read the tree

Set how many of each item are in the bag, choose replacement, then read the auto-calculated probabilities. Click any blue branch fraction to edit a first-stage probability by hand.

Use your tree. Type the probability as a fraction in its simplest form, or as a decimal. The app marks it and tells you exactly which branches to multiply or add.
