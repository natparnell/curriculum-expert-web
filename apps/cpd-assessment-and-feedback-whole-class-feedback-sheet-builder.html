    Why this works: whole-class feedback (sometimes called a feedback sheet or marking codes approach) replaces individual written comments with one shared response. You read the class set once, note patterns, then teach to the patterns. The DfE workload reduction work and the EEF guidance both point the same way: feedback should redirect effort onto pupils and away from repetitive teacher writing. Sort well and the meter rewards a sheet that is efficient and still precise.
