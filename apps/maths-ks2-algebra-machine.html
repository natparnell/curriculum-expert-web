How a function machine works

A function machine takes an input, applies a rule, and gives an output. The rule here is "multiply, then add", for example "multiply by 3, then add 2".

  • Find the output: put the input in, follow the rule. Input 5 gives 5 × 3 = 15, then 15 + 2 = 17.
  • Find the rule: you are given pairs of inputs and outputs. Look at how the output grows each time the input grows by 1: that gap is the multiplier.
  • Find the input: you know the output, so undo the rule. Working backwards from 17: subtract 2 to get 15, then divide by 3 to get 5.

If we call the input x and the output y, the rule "multiply by 3, then add 2" is the formula y = 3x + 2. The x is an unknown: a letter that stands for a number we do not know yet.
