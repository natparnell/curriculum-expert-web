Search and Sort Visualiser

Teacher aim: pupils compare linear search and binary search on the same sorted array, count comparisons, and complete a live trace table to reason about efficiency (Big O intuition).

1 Build a sorted array and pick a target

Binary search only works on a sorted array, so the array is always kept in order. Choose a size, generate values, then pick the value to search for.

Tip: pick a value that is in the array, or use "Pick a missing value" to see how each algorithm reports "not found".
