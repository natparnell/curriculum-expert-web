💡 Binary Bulb Board

Teacher aim: Pupils represent numbers, text and images in binary. Toggle eight bulbs as place values to see denary and hexadecimal update live, then paint a black-and-white image to discover how pictures are stored as bits.

One byte, eight bulbs

Each bulb is a bit. Switch a bulb on and you add its place value. Watch the totals change.

Binary
00000000
Denary (base 10)

Hexadecimal (base 16)

Tap or click a bulb. With keyboard: Tab to a bulb, then Space or Enter to toggle.

Hit the target

Set the bulbs to make the target number, then press Check. The app tells you exactly which place value to fix.

An image is bits too

Paint a black-and-white picture. Each square is one pixel: black is 1, white is 0. Each row of eight pixels is one byte, exactly like the bulb board.

Drag with finger or mouse to paint. With keyboard: Tab to a pixel, then Space to flip it.

How this image is stored

Each row as a byte

A 1-bit black-and-white image needs only one bit per pixel. A colour photo uses many more bits per pixel, so its file is much bigger.
