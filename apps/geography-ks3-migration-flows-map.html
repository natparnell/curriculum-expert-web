🌍 Migration Flows Map

Teacher aim: Pupils investigate how push and pull factors drive migration. Drag factors onto the source and destination countries, then read how the migration flow and the impacts on each country change.

Factor tray

Drag a card onto a country, or select it and press a number key. Push factors push people away; pull factors draw people in.
