My Digital Drawer 🗄️

Computing · Key Stage 1 · Using technology purposefully: create, save and find your work

Teacher aim: Pupils create a picture, give it a name, choose a folder and save it, then find and open it again later. This models the save, name, retrieve loop that underpins all digital file handling.
1 Make a picture
2 Name and save
3 Find it again

2. Name your work and save it

A good file name tells you what is inside. Choose a folder so you can find it later.
