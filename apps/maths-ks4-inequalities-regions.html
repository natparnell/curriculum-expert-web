    Exam-style practice

    How this works: conventions, method and exam tips

    Tool A — graphing regions

    • Each inequality is rearranged to a boundary line. The line is drawn dashed for a strict inequality (< or >) because points on the line are not included, and solid for an inclusive one (≤ or ≥) because points on the line are included.
    • The feasible region is the set of points that satisfy every inequality at once. Use the toggle to switch between the two common GCSE conventions: shade the unwanted side of each line and leave the feasible region clear (the exam-standard method), or highlight the feasible region directly.
    • To decide which side of a line to keep, substitute a test point (often the origin) into the inequality. If it is true, keep the side containing that point.

    Tool B — solving inequalities

    • Linear: solve like an equation, but reverse the inequality sign whenever you multiply or divide both sides by a negative number.
    • Quadratic: rearrange to = 0, factorise or use the quadratic formula to find the critical values (the roots), then sketch the parabola. A positive coefficient of x² gives a U shape; the curve is below the axis between the roots and above it outside the roots.
    • Read the solution off the sketch: > 0 wants the parts above the axis; < 0 wants the part below. This is why x² - x - 6 > 0 gives two separate pieces (x < -2 or x > 3) while < 0 gives a single interval (-2 < x < 3).

    Set notation reminder

    • {x : x > 3} reads "the set of all x such that x is greater than 3".
    • Two separate pieces are joined with the union symbol ∪, for example {x : x < -2} ∪ {x : x > 3}.
