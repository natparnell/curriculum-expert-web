🐝 Bee-Bot Floor Path

Teacher aim: children give simple instructions to a programmable toy and predict the result. This builds early control and cause-and-effect: each button press makes the bee do one clear thing.

🧩 Build the path

Drag an arrow into the path, or tap an arrow to add it to the end. The bee always starts in the bottom corner facing up.

📜 The bee's path

    Before you press Go: ask the class, "Which flower will the bee land on?" Then watch it buzz the route step by step.

    🌻 The flower mat

    Build a path on the left, then press Go to watch the bee buzz along.
