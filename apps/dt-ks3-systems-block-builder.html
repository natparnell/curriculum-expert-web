System canvas

Keyboard: Tab to a block, arrow keys to move it, Enter to draw a wire to the next stage, Delete to remove it.

Every electronic system is three blocks

  • Input: a component that senses a change, for example a light sensor, a temperature sensor or a push switch. It turns a real-world quantity into a signal.
  • Process: a programmable component, usually a microcontroller, that takes the input signal, applies a rule (for example "if it is dark, switch on"), and decides the output.
  • Output: a component that acts on the world, for example an LED, a buzzer, or a motor that drives a mechanism.

Changing the rule in the process block changes the behaviour of the whole system without changing any wiring. That is the power of a programmable component.

Folding in mechanisms

  • An output motor rarely drives the load directly. A mechanism changes its speed, force or type of motion.
  • The DC motor output in this app lets you add a gear ratio. A larger gear-down ratio gives more turning force (torque) but slower output speed.
  • This links electronic control to mechanical advantage: the process decides when the motor runs, the mechanism decides how it moves the load.

Words to use accurately

  • Signal: the information passing along a wire between blocks.
  • Threshold: the value at which the process changes its decision.
  • Feedback (control): when the output state is fed back to influence the next decision. Not used in these simple open-loop systems, but worth naming.
  • Gear ratio: the number of input turns needed for one output turn. 4:1 means four motor turns for one output turn.
