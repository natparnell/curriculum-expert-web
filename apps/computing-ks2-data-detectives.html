Data Detectives 🔍

Computing · Key Stage 2 · Data and information

Teacher aim: Pupils collect class data into a spreadsheet, sort and chart it, then use reasoning to spot a value that cannot be true.
1 Collect data
2 Sort the sheet
3 Make a chart
4 Spot the fake

Step 1: Collect the data spreadsheet

Class 4B measured how many minutes each person walked to school. Add the readings into the spreadsheet, one row at a time. We have started you off with some demonstration readings.

Tip: a few sensible readings are already in the sheet below.

#NameMinutesRemove

Step 2: Sort the spreadsheet

A real spreadsheet can put the rows in order for you. Click a column heading to sort by it. Sorting helps you compare values and notice anything odd at the very top or bottom.

# Name Minutes

Sort by Minutes to put the times in order. Watch where the biggest number ends up.

Step 3: Present it as a bar chart

Charts show the data as a picture. Press the button to build the bars. A very long bar can be a clue: it might be a reading that is too big to be true.

Each bar is one person. The longest bar shows the highest reading.

Step 4: Spot the value that cannot be true 🕵

One reading is not plausible. That means it does not make sense in real life. Use your reasoning, not just the chart.

Detective clue: A walk to school usually takes a few minutes up to about an hour. There are only 60 minutes in an hour. Which reading could not really be a walk to school?
