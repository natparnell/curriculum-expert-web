How the maths works (for teachers and curious pupils)

Direct proportion. Two quantities are in direct proportion when one is always the same multiple of the other. In the ratio table the same multiplier scales both rows, so the ratio between the rows never changes. Doubling one doubles the other.

The double number line shows both quantities side by side on a common scale. Sliding to a point that is, say, three fifths of the way along multiplies both lines by the same fraction. This is why best-buy comparisons work: scale each option to the same amount, then compare.

Inverse proportion is different. When one quantity goes up, the other goes down so that the product stays the same (for example, workers and time: more workers, less time). Here you multiply one row and divide the other by the same factor.

  • Find the multiplier: divide the target known value by the starting known value.
  • Apply the same multiplier to the partner row (direct) or its reciprocal (inverse).
  • Check: does the answer keep the original ratio sensible?
