Oracy and Talk Protocol Planner

Teacher aim: design a structured classroom discussion. Choose a talk protocol, assign talk roles, select sentence stems and set the oracy challenge so every pupil knows how to take part.

Give the talk a clear, debatable focus. A genuine question beats a topic label.

Question check

Type a discussion question above and this will tell you whether it is open enough to sustain talk.

2Choose a talk protocol

The structure that moves pupils from private thinking to public talk. Select one.

3Assign talk roles

Drag roles into the group, or use the keyboard (Tab to a role, then press Enter to move it). Aim for two to four roles so talk stays balanced.

4Pick sentence stems

Stems give pupils a way in. Choose three to six across different talk moves so the discussion can build, challenge and connect.
