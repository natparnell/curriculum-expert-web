How the maths works (for the board)
  • Pixels: total pixels = width × height. Halve both sides and you quarter the pixel count.
  • Colour depth: a depth of n bits gives 2n colours. 1 bit = 2 colours (black and white), 8 bits = 256 colours, 24 bits = true colour (about 16.7 million).
  • File size (bits): width × height × colour depth.
  • File size (bytes): divide the bits by 8 (there are 8 bits in a byte). 1024 bytes = 1 KB.
  • Compression: lossless methods like RLE rebuild the exact image. Lossy methods (such as JPEG) throw away detail the eye barely notices to save more space.
