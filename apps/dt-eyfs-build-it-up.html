For the grown-up: how to use this

Curriculum link (EYFS, Expressive Arts & Design / Understanding the World; foundations for Design & Technology): constructing and joining materials to make stable structures, and explaining simple ideas about why some structures stand and some fall.

  • Build together: Tap blocks to add them, then drag to stack. Try a tower (stacked up) and a bridge (two towers with a wide block across the top).
  • Predict first: Before pressing wobble, ask "Do you think it will stay up? Why?"
  • Test: The wobble test shakes the structure. A wide base and good stacking keep it standing; a narrow base or wobbly tower may topple.
  • Key vocabulary: stack, base, wide, narrow, balance, stable, topple, strong.
  • Accessibility: fully keyboard operable. Tab to a block, then use arrow keys to move it; Space tests; Delete removes the selected block. All actions are announced for screen readers.
