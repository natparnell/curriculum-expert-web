How does this work? (the key idea)

A message is too big to send in one go, so it is chopped into small packets. Each packet carries a header: the destination address, a sequence number (so the message can be rebuilt in order), and the data. Routers pass each packet on towards its destination, choosing the next working link. If a router fails, the others simply route around it. This is packet switching, and it is why the internet keeps working even when parts break.
