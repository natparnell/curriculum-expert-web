Why spacing and interleaving? (the evidence in brief)

Spacing means revisiting a topic after a gap rather than massing all practice together. The forgetting that happens in the gap is the point: retrieving something you have started to forget strengthens the memory more than re-reading it while it is still fresh. This is the spacing effect, one of the most robust findings in cognitive science.

Interleaving means mixing several topics within a single retrieval session rather than blocking one topic at a time. It feels harder and progress feels slower, but it forces pupils to discriminate between problem types and to retrieve the right strategy, which improves long-term transfer.

  • Blocked and massed: A A A then B B B in adjacent lessons. Feels fluent, fades fast.
  • Spaced and interleaved: each topic taught once, then revisited in later lessons, with several topics revisited in the same lesson.

This tool is a planning aid for distributed retrieval. It complements a quiz generator: the quiz writes the questions, this decides when each topic should reappear.
