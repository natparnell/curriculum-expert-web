Prerequisite Knowledge Mapper

Teacher aim: before teaching a new topic, identify the prior knowledge it quietly assumes, then run a 60-second diagnostic so you teach into a secure foundation rather than over a gap.

  1. Choose the new topic
  2. Sort what it depends on
  3. Check the build is sound
  4. Run the diagnostic

1. Which new topic are you about to teach?

Pick a topic. The mapper shows you ten candidate facts and skills. Only some are genuine prerequisites the lesson assumes pupils already hold.

2. Sort the candidates: is it a prerequisite or not?

Drag each card into a column, or use the keyboard (see note below). A genuine prerequisite is knowledge the new topic directly builds on; pupils must already hold it for the new learning to land. The "not required" pile is for related-but-not-needed knowledge that often sneaks into planning.

Not required for this topic

Keyboard: Tab to a card, then press P for Prerequisite, N for Not required, or U to return it to Candidates.

3. Diagnostic: is each prerequisite actually secure?

The mapper has written one quick check per confirmed prerequisite. Answer as a pupil would (try it cold), or use this with a class as a hinge-question starter. Each item names the prerequisite it probes so a wrong answer tells you exactly what to reteach.
