Moderation Reliability Checker

Teacher aim: standardise a marking team before data entry. Set each marker's score on common scripts, then check the team for drift, leniency or harshness, and inconsistency so the agreed mark is reliable.

STEP 1 Choose a common script

Every marker in a moderation panel marks the same scripts. Pick a script, read it, then record what each marker awarded out of 10.

What is a "true" agreed mark?
In real moderation the lead moderator holds an agreed mark for each common script, set against the mark scheme during standardisation. This app uses a fixed agreed mark per script (shown after you check reliability) so each marker's distance from the standard can be measured. The further a marker sits from the agreed mark, the more they drift.

STEP 2 Check reliability

Set scores for all three scripts, then run the check. The panel flags any marker who is reliable, drifting, lenient, harsh or inconsistent before any data is entered.
